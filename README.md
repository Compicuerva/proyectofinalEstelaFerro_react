# proyectofinalEstelaFerro_react

Trabajo final creando una aplicacion ReactJS utilizando Vite. 

1. Se instalo React en VSC y con JSX se crearon componentes funcionales
2. Se implemento la gestión de estado utilizando useState.
3. Se desarrollo rutas dinámicas y protegidas para una navegación fluida y segura.
4. Ciclo de vida con useEffect: Realizar peticiones a APIs y manejar efectos secundarios.
5. React Router: Configuración de rutas, navegación y rutas protegidas.
6. Context API: Crear y consumir contexto para el manejo del estado global (ej. carrito, autenticación).
7. Se construyo un formulario con validación y manejo de errores, además de la autenticación de usuarios.
8. Se desarrollo un CRUD básico para gestionar productos dentro de un proyecto de e-commerce de joyas.
9. Se estilizo la aplicación utilizando la librería de Bootstrap y styled-components.
10. Diseño mobile-first y adaptabilidad de la aplicación, con mejores prácticas de diseño UI/UX.
